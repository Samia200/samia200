# Samia GitHub Profile — Setup

This pack is made for the GitHub profile repository:

`https://github.com/Samia200/Samia200`

## Install

1. Open your `Samia200/Samia200` repository.
2. Upload everything from this pack while keeping the folders exactly as they are:
   - `README.md`
   - `assets/samia-cinematic.svg`
   - `.github/workflows/snake.yml`
   - `.github/workflows/profile-3d.yml`
3. Commit the files to the default branch.
4. Open **Settings → Actions → General**.
5. Under **Workflow permissions**, choose **Read and write permissions**, then save.
6. Open the **Actions** tab.
7. Run **Generate Contribution Snake** manually once.
8. Run **GitHub Profile 3D Contributions** manually once.
9. Refresh your GitHub profile.

## What moves

- The cyber-anime banner uses native SVG animation.
- The introduction text is an animated typing SVG.
- The contribution snake is regenerated daily.
- The 3D contribution graph is regenerated daily.
- GitHub statistics update automatically.

## Important

GitHub README pages do not run custom JavaScript or real WebGL scenes. This design uses animated SVGs and generated GitHub Actions, which are compatible with GitHub profile READMEs.
